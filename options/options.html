About

Chromium Search Bar v1.0.0

A dedicated search bar extension for Chromium-based browsers that brings back the classic search bar experience.

Features:

  • Multiple search engine support (Kagi, Google, DuckDuckGo, Yahoo, Ecosia)
  • Live search suggestions as you type
  • Quick access via keyboard shortcut or omnibox keyword
  • Modern, dark-themed interface
